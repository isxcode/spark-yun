##### 解决方案

```text
麒麟系统本地默认的java版本不匹配，重新指定一个JAVA_HOME，推荐使用java1.8
```

```bash
vim /opt/zhiqingyun/conf/zhiqingyun-env.sh
```

```bash
export JAVA_HOME=/zulu/java
```
