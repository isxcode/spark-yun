#### 问题1

```wikitext
作业运行卡住，数据库排查发现大量待运行的定时任务
```

![20260305190844](https://img.isxcode.com/picgo/20260305190844.png)

##### 解决方案

```wikitext
默认使用H2数据库，服务器的最大打开文件数限制，默认为1024，太小会导致H2无法查询数据。
```

##### 临时配置

```bash
ulimit -n 65535
```

##### 永久配置

```bash
vim /etc/security/limits.conf
```

```bash
* soft nofile 65535
* hard nofile 65535
```
