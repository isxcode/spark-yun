## 使用官网安装包离线部署

#### 下载安装包或者源码编译

官网链接：https://zhiqingyun.isxcode.com

![20240427182151](https://img.isxcode.com/picgo/20240427182151.png)

#### 离线安装java环境

以CentOS7.9系统为例

[点击下载官网OpenJdk](https://cdn.azul.com/zulu/bin/zulu8.94.0.17-ca-jdk8.0.492-linux_x64.tar.gz)

```bash
# 推送到服务器
scp zulu8.94.0.17-ca-jdk8.0.492-linux_x64.tar.gz root@47.116.163.22:/tmp/
```

⚠️ **注意**

> 一定要在~/.bashrc中配置JAVA_HOME环境变量

```bash
tar -vzxf /tmp/zulu8.94.0.17-ca-jdk8.0.492-linux_x64.tar.gz -C /opt
ln -s /opt/zulu8.94.0.17-ca-jdk8.0.492-linux_x64 /opt/java
tee -a ~/.bashrc <<-'EOF'
export JAVA_HOME=/opt/java
export PATH=$PATH:$JAVA_HOME/bin
EOF
source ~/.bashrc
java -version
```

⚠️ **注意** 

> 服务器的最大打开文件数限制数，默认为1024，太小会导致作业执行速度变慢，建议修改为65535。

```bash
# 临时配置方式
ulimit -n 65535
```

#### 上传并解压安装包

```bash
# 上传
scp zhiqingyun.tar.gz root@47.116.163.22:/tmp/
# 解压到/opt下
tar -vzxf /tmp/zhiqingyun.tar.gz -C /opt
```

#### 修改管理员登陆密码

```bash
vim /opt/zhiqingyun/conf/application-local.yml
```

```yml
server:
  port: 80 # 修改端口号
   
isx-app:
  admin-passwd: admin1234 # 修改管理员密码
```

#### 启动项目

```bash
bash /opt/zhiqingyun/bin/start.sh
```

#### 访问项目

- 访问项目: http://47.116.163.22 
- 管理员账号：`admin` 
- 管理员密码：`admin1234` 

#### 配置至轻云开机自启（可选）

```bash
chmod a+x /opt/zhiqingyun/bin/start.sh
chmod a+x /opt/zhiqingyun/bin/stop.sh
```

#### 配置自启文件

```bash
vim /usr/lib/systemd/system/zhiqingyun.service
```

> 配置启动脚本路径和pid文件路径和启动用户

```bash
[Unit]
Description=Zhiqingyun Service unit Configuration
After=network.target

[Service]
Type=forking

# 添加工作目录
WorkingDirectory=/opt/zhiqingyun

# 添加 JAVA_HOME 和完整 PATH
Environment="JAVA_HOME=/opt/java"
Environment="PATH=/opt/java/bin:/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin"

ExecStart=/opt/zhiqingyun/bin/start.sh --print-log="false"
ExecStop=/opt/zhiqingyun/bin/stop.sh
ExecReload=/opt/zhiqingyun/bin/start.sh --print-log="false"
PIDFile=/opt/zhiqingyun/zhiqingyun.pid
KillMode=none
Restart=always
User=root
Group=root

[Install]
WantedBy=multi-user.target
```

#### 重载服务

```bash
systemctl daemon-reload
```

#### 设置开机自启

```bash
systemctl enable zhiqingyun
```

#### 相关操作命令

```bash
# 启动至轻云
systemctl start zhiqingyun

# 查看至轻云状态
systemctl status zhiqingyun

# 停止至轻云
systemctl stop zhiqingyun

# 重启至轻云
systemctl restart zhiqingyun
```
