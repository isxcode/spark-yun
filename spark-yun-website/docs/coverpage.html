产品优势

  1. Docker一键部署
    快速地将应用程序打包到容器中，并且可以在不同的环境中快速部署。
  2. 系统无缝集成
    快速响应业务需求，缩短了系统开发和部署的时间，提高了业务的敏捷性。
  3. 代码开源
    让全球的开发者协同开发，共同完善代码，提高代码的质量和稳定性。经过众多开发者和用户的测试和使用，可以更加可靠和稳定。

  1. 基础数据源
    包括：MySQL、Oracle、SQL Server等。
  2. 结构化数据源
    包括：Hive、PostgreSQL、DB2、Hana SAP、达梦数据库等。
  3. MPP数据源
    包括：Tidb、Oceanbase、StarRocks、Doris等。

  1. 3种基础作业
    包括：Jdbc查询作业、SparkSql作业、Jdbc执行作业。
  2. 数据采集
    包括：CSV文件采集到Hive、Excel文件采集到Hdfs等。
  3. 数据查询
    支持大数据秒速查询返回，基于Spark两种模式：Per-job模式和Session模式。
  4. 其他作业
    包括：Shell脚本、ETL作业、Python脚本、自定义Spark函数、自定义Spark插件等。

  1. 数据模型
    支持数据分层、数据域、业务过程等功能。
  2. 数据字典
    可自定义数据字典，对大数据中的字段进行字典翻译。
  3. 数据指标
    可快速对数据进行归总和分析，且支持可视化图形展示。
  4. 数据标签
    数据支持打标签功能。

  1. 可视化配置
    提供可视化配置接口，零代码亦可配置开放Api接口，为其他服务提供数据。
  2. 安全加密
    接口具备Token身份校验机制，或配置客户自定义身份认证逻辑。
  3. 数据脱敏
    对特殊数据进行脱敏管理，根据不同用户权限访问不同加密数据。
  4. 低延迟响应
    基于Spark实现数据查询，全内存查询，提供超快查询能力。

  1. 逻辑作业流
    作业流支持逻辑节点，包括循环节点、判断节点、分流节点、归并节点等。
  2. 实例查询
    作业运行实例化，提供作业实例界面UI。
  3. 基线管理
    配置作业基线，支持短信提醒，邮箱提醒，消息中心提醒等方式。

应用场景
汽车行业

利用大数据来提高汽车的性能和用户体验。 智能车辆拥有大量的传感器，可以收集各种数据，包括车速、加速度、方向盘转向角度、制动力、电池温度等等。 这些数据被传送到大数据的云平台上，然后被用来进行数据分析和改进汽车的性能和用户体验。

半导体行业

半导体制造公司利用大数据来改进制造过程和供应链管理。 工厂和供应链拥有大量的传感器和监控系统，可以收集各种数据，包括生产效率、质量控制、零部件供应等等。 这些数据被用来进行制造过程和供应链优化，从而提高效率和减少成本。

用户画像

利用大数据来构建用户画像，更好地了解客户需求和购物行为。 收集并分析各种数据，包括浏览历史、购买记录等等。 通过对这些数据的分析，能够更好地了解客户的购物偏好、兴趣爱好、年龄段等等，从而为客户提供更加个性化的商品推荐和营销策略。

BI报表

利用大数据BI报表，分析车站客流、列车运行、货运等多种数据来源，以更加准确地了解客流情况、车辆运行情况等等。 BI报表能够通过实时数据分析、数据可视化等方式，为管理者提供直观的数据展示和分析，从而为企业提供更加精准的管理支持。
