### 特色

- **代码开源**：允许用户自定义二次开发
- **高可用**：支持高可用集群部署，提高用户环境抗灾能力
- **多租户**：一套系统，租户数据隔离，多部门共同使用
- **云原生**：支持docker快速部署，只需一行命令即可使用
- **私有化部署**：用户可内网部署，提高用户数据安全性
- **spark纯原生**：支持spark官网的所有原生用法和规则
